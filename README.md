# DeepinkApp Tap

## How do I install these formulae?

`brew install deepinkapp/tap/<formula>`

Or `brew tap deepinkapp/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "deepinkapp/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
